# Exercício Menu Interativo - Jogo de Adivinhação

## Descrição
Projeto pessoal desenvolvido durante o curso de programação em C.
Um jogo de adivinhação de números com menu interativo usando estruturas switch/case.

## Funcionalidades
- Menu principal interativo
- Jogo de adivinhação de números (0 a 9)
- Número gerado aleatoriamente a cada partida

## Tecnologias utilizadas
- Linguagem C
- Bibliotecas: stdio.h, stdlib.h, time.h

## Como executar
1. Compile o arquivo com GCC:
gcc menu_interativo.c -o menu_interativo.exe
2. Execute o programa:
./menu_interativo.exe
